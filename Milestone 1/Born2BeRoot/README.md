DESCRIPTION

This project has the goal of learning how to set up a virtual machine and use other softwares, also it forces you to do all your tasks through the shell so you learn to work at a low level.
I chose Debian because i thought it was more easier to setup and learn basic mechanics within a new software and learn well how to setup and configure a Virtual Machine.

INSTRUCTIONS

Inicialize Virtual Box and initialize the machine, followed by unlocking the disk and inserting your credentials.

PROJECT DESCRIPTION

Debian is a easier or (begginer friendly) software for begginers, it is easier to setup and configure compared to Rocky which is a more complex software with its focus on a more company level, it offers bigger configuration and bigger security at the cost of higher complexity. AppArmor and SELinux are 2 security technologies that their complexity is very similar to Debian and Rocy in which AppArmor is easier to setup and more begginer friendly and SELinux has much more security control at the cost of complexity.
The difference between UFW and firewalld is that UFW is much more easier to use but starts to decay on control and readability on a larger scale while firewalld can scale much more and offer more configuration at the trade-off of higher complexity.
VirtualBox is a more feature rich and compatible general software while UTM is more simple for beginners and has its strength of beeing more optmized for Apple silicon models.
