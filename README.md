# 42 Common Core Projects

A collection of projects completed during the **42 Common Core** curriculum. This repository showcases my progress throughout the program, covering low-level programming, algorithms, system administration, software architecture, and problem-solving.

Each project was developed following the strict coding standards and peer-review process of the 42 School.

## Projects Included

### Milestone 0

#### Libft

A custom implementation of the C Standard Library.

**Topics covered**

* Memory management
* String manipulation
* Linked lists
* Makefiles
* Code organization

#### get_next_line

A function capable of reading a file descriptor line by line while handling multiple file descriptors efficiently.

**Topics covered**

* Static variables
* Buffer management
* File I/O
* Dynamic memory allocation

#### ft_printf

A recreation of the standard C `printf()` function supporting multiple format specifiers.

**Topics covered**

* Variadic functions
* Format parsing
* Number conversions
* Memory management

---

### Milestone 1

#### Born2BeRoot

A Linux system administration project focused on creating and securing a virtual machine.

**Topics covered**

* Debian
* Virtual Machines
* SSH
* UFW Firewall
* User and group management
* Password policies
* System monitoring

#### Push Swap

An algorithmic sorting project where a list of integers must be sorted using only two stacks and a limited set of operations.

**Topics covered**

* Algorithms
* Data structures
* Stack manipulation
* Sorting optimization
* Complexity analysis
